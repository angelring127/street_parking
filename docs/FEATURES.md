# 기능 상세

## 현재 사용자 기능

### 지도 탐색

- Leaflet 기반 지도 사용
- 가격이 표시된 커스텀 마커 사용
- 지도 이동과 줌 동작 지원
- 선택된 미터로 지도 중심 이동

### 리스트 탐색

- 카드형 리스트 UI
- 초기 20개부터 보여주고 스크롤 시 추가 로드
- 카드 클릭 시 지도 이동
- 상세 요금표 토글 가능

### 검색

- OpenStreetMap Nominatim 기반 주소 자동완성
- 입력 500ms debounce
- 캐나다로 제한된 검색
- Vancouver 주변 `viewbox` 우선 검색

### 현재 위치

- 브라우저 Geolocation API 사용
- 위치 성공 시 해당 좌표로 지도 이동
- 거리 정렬 및 거리 표시 활성화

### 필터

- 요일 선택
- 시간 선택
- 최대 요금 필터
- 신용카드 결제 가능 여부 필터

### 정렬

- 가격 낮은 순
- 가격 높은 순
- 거리 순

데스크톱 필터 패널 기준으로는 위 3가지를 지원합니다.

### 시간 기반 요금 표시

- 사용자가 선택한 요일/시간 기준으로 리스트 요금 표시
- 지도 마커 라벨에도 선택한 요일/시간 기준 요금 표시
- 상세 요금표에서 각 시간대 제한 시간도 함께 표시

### 내비게이션 링크

- Google Maps 링크 제공
- Apple Maps 링크 제공

### 다국어

- 한국어
- 영어

### 반응형 UI

- 데스크톱: 필터 / 지도 / 리스트 3영역 레이아웃
- 모바일: 지도 / 리스트 탭 전환
- 모바일: 햄버거 메뉴 안에 별도 필터 UI 존재

## 현재 코드 기준 구현 메모

### 지도와 리스트 동기화

- 지도 마커 클릭 시 리스트의 해당 카드로 스크롤
- 리스트 카드 클릭 시 지도 중심 이동
- 모바일에서는 리스트에서 "지도에서 보기" 버튼으로 지도 탭 전환 가능

## 현재 남아 있는 구현 이슈

- 가격 필터는 선택 시간 기준이지만 가격 정렬은 실제 현재 시간 기준
- 지도 팝업 요금은 선택 시간이 아니라 실제 현재 시간 기준
- 모바일 정렬 메뉴 값이 타입 정의와 일부 맞지 않음
